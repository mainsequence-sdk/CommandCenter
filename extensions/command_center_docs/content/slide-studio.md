# Slide Studio

Slide Studio is the presentation-focused part of the Workspaces application. It uses the same
workspace foundation, but organizes content as slides instead of one continuous operating canvas.

## When To Use It

Use Slide Studio when the outcome is a deck, briefing, committee pack, review pack, or another
presentation that needs a clear slide-by-slide narrative.

Use regular Workspaces when the goal is an interactive operating view that stays open for day-to-day
monitoring or investigation.

## Core Workflow

1. Open the `Slide Studio` surface.
2. Create a new deck or open an existing one.
3. Add slides and place widgets on each slide.
4. Use notes, charts, tables, and metrics to build the story.
5. Preview the deck in presentation mode.
6. Export to PDF when the deck is ready to distribute.

## What Makes It Different

- The workspace list is scoped to slide-studio decks.
- The layout is slide-oriented instead of one long canvas.
- Presentation mode is part of the normal workflow.
- PDF export is a first-class action for completed decks.

## Good Practice

- Keep one primary idea per slide.
- Use consistent slide titles so the deck is easy to scan.
- Preview in presentation mode before exporting.
- Keep supporting detail in backup slides instead of overloading the main story.
