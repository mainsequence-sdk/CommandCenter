# Main Sequence Foundry

Main Sequence Foundry is the operational application for Main Sequence backend resources. It is the
place to work with projects, jobs, data resources, and deployment-oriented infrastructure records.

## Main Areas

- `Projects` for project-level workflows.
- `Jobs` for cross-project job browsing and run inspection.
- `Resources` for release-style resource surfaces such as Streamlit.
- `Data` for buckets, data sources, data nodes, and tables.
- `Deployment Services` for service and infrastructure-oriented records.

## Typical Use

Use Foundry when the work is about managing backend objects, infrastructure-backed resources, or
project runtime operations rather than building a workspace presentation.

## Current Surface Shape

Foundry is a multi-surface application. Navigation is grouped into operational areas so users can
move between projects, jobs, resources, and data without leaving the same product domain.
