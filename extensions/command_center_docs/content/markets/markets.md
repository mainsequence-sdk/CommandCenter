# Main Sequence Markets

Main Sequence Markets is the market-facing application for asset registries, portfolio structures,
and related operational reference data.

## Main Areas

- `Assets` for master asset lists and asset categories.
- `Portfolios` for virtual funds, portfolios, and portfolio groups.
- `Settings` for supporting configuration surfaces such as instruments and execution venues.

## Typical Use

Use Markets when the work is centered on market entities, portfolio structure, or market-specific
reference data rather than on backend infrastructure or workspace authoring.

## Current Surface Shape

Markets is organized by functional areas instead of one flat list. That keeps asset, portfolio, and
settings workflows separate while still living in one market-focused application.
