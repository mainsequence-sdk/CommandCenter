# Workspaces Content

This directory holds the markdown pages for the `Workspaces` documentation section and its ordered
submenu pages.

## Pages

- `workspaces.md`: workspace canvas, graph, settings, and general authoring flow.
- `widgets.md`: widget catalog and saved-widget usage.
- `reference-variables.md`: live references from one widget value into another widget setting.
- `slide-studio.md`: presentation workflows built on the workspace model.
