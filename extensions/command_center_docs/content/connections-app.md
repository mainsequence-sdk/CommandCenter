# Connections

Connections is the platform-admin application for configuring and inspecting data-source access in
Command Center.

## Main Areas

- `Add A New Connection` for browsing available connection types and creating configured instances.
- `Data Sources` for reviewing configured backend-owned data-source instances.
- `Explore` for testing a configured source and running live queries against it.

## Typical Use

Use the Connections application when you need to register, inspect, or test the underlying data
sources that widgets and application surfaces depend on.

## Access Model

Connections is a platform-admin application. It is not the place where normal end users consume
data inside workspaces. It is the place where administrators prepare and inspect the connection
layer that those user-facing workflows rely on.
