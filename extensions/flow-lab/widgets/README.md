# Flow Lab Widgets

This folder contains widget modules owned by the external Flow Lab extension.

## Modules

- `order-book/`: Level II and depth-style order-book widgets that remain available on disk for
  Flow Lab surfaces.

## Maintenance Notes

- Keep Flow Lab-owned widget code here instead of under `src/widgets/`.
- Each widget folder should keep its own `README.md` with entry points and maintenance notes.
