## How To Extend Operations Docs

- Put hosting, deployment, and environment constraints here when they affect the repository as a
  whole.
- Keep product-specific rollout notes out of this section unless they change the shared deployment
  contract.

## Guidelines

- Document the operational consequence, not just the mechanism.
- Be explicit when a deployment rule is a hard requirement instead of a suggestion.
