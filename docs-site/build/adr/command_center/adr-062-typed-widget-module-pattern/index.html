ADR 062: Typed Widget Module Pattern

  • Status: Proposed
  • Date: 2026-05-19

Capture the intent to move toward a typed widget module pattern that groups widget definition, model, settings, README, USAGE_GUIDANCE, tests, and registry validation under one enforced structure, instead of moving widgets toward inheritance-heavy class-based modules.
